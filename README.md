### Dictionary for gramps

This gramplet will show you the glossary for Gramps.  
It will show the term, as it appears in the python code, and with two additional languages.  
The languages is provided in a separate csv file for each lanuage.  

Double clicking will take you to the Gramps wiki explanation of the term.

Clicking one of the language columns and start typing will do a search in that column based upon your typing
